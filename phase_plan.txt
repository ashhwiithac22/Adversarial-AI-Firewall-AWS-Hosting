🤖 PHASE 5: DUAL-INPUT NEURAL NETWORK (FIREWALL MODEL)
🎯 Goal
Build model that takes both image (128x128) and 15 features to detect attacks.

✅ Steps to Achieve
5.1 CNN Branch (Image Features)
Input: 128x128x3 RGB

Conv2d(3→32, 3x3) → BatchNorm → ReLU → MaxPool(2x2)

Conv2d(32→64, 3x3) → BatchNorm → ReLU → MaxPool(2x2)

Conv2d(64→128, 3x3) → BatchNorm → ReLU → AdaptiveAvgPool(1x1)

Output: 128 features

5.2 MLP Branch (Metadata Features)
Input: 15 features

Linear(15→64) → ReLU → Dropout(0.3)

Linear(64→32) → ReLU

Output: 32 features

5.3 Combined Classifier
Concatenate CNN (128) + MLP (32) = 160

Linear(160→128) → ReLU → Dropout(0.4)

Linear(128→64) → ReLU

Linear(64→1) → Sigmoid

Output: Probability (0=Clean, 1=Attack)

5.4 Model Definition
Create models/firewall/firewall_model.py with class AdversarialFirewall

Add device handling and forward pass

✅ Phase 5 Completion Criteria
Model code complete

Model can accept (batch,3,128,128) image tensor and (batch,15) feature tensor

Output shape: (batch,1) with values 0-1

~500K parameters
